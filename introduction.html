Fantasy-Chess Documentation Help

Introduction

Fantasy-Chess ist a PvP, turn-based action game, in which players command characters to fight against each other in order to claim victory.

Getting Started

For the best experience please install Jetbrains IntelliJ IDEA or download one of the existing binaries under releases.

Running the Project

Downloaded binaries

If you have running binaries, you can run the client by hosting it on any webserver. For local use we recommend using npx serve.

Running the Client
npx serve dist
Running the Server
java -jar fantasychess-server-{version}.jar

Building from source

If you do not have existing binaries, you can build them by using the following commands, once you have cloned the project:

Building the Client
cd fantasychess-client/ && ./gradlew dist
Building the Server
cd fantasychess-server/ && ./gradlew bootJar

Development

Ensure you've installed Java 21 or higher, along with either Jetbrains IntelliJ IDEA or a local installation of Gradle 8.9.

  1. Clone the repository from https://github.com/B-Team-Organisation/Fantasy-Chess

  2. Open up the project with IDEA

  3. Add the click on File > Add module from existing sources and add the common module

  4. Repeat the previous step for the fantasychess-server and fantasychess-client module IN THAT ORDER.

You are now ready to start working on the Project.
