Grid Service

Author: Lukas Walker

The Grid Service wraps a Grid Model and hosts methods providing information about the Grid Model or to manipulate it. It only holds a single Grid Model:

private final GridModel gridModel;

The following tasks can be accomplished using the Grid Service:

Information retrieval

  • Retrieve a character from a specified position

  • Retrieve the Tile Model of a specific position

  • Retrieve the dimensions of the Grid Model

  • Retrieve the underlying Grid Model

  • Check if a given position is a valid position on the grid

Grid Manipulation

  • Swap the positions of two characters on the grid

  • Move a characters from a tile to another

  • Place a character on a specific tile

  • Remove a character from a specific tile

  • Flag a number of rows as start tiles:

For implementation details, please check out the JavaDoc
